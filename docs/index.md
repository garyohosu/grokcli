# GrokCLI Documentation

Welcome to the official documentation for **GrokCLI**, an AI-powered terminal assistant built on the Grok API.

This documentation is organized into the following sections:

- **Quickstart Guide** — Installation & first steps
- **Agent Mode** — Overview of the autonomous workflow system
- **Specifications** — Internal technical details of the agent system
- **Architecture** — High-level system design

All diagrams are written in Mermaid format and render natively on GitHub.
